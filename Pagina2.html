Ingenria de Software

Los ingenieros de software se encargan de analizar, diseñar, crear y probar los sistemas informáticos y de software. Además, escriben programas de software para satisfacer las necesidades de un cliente o para resolver un problema particular.


"¿qué conocimientos debe tener un ingeniero de software? Precisamente esto es lo que busca responder el Software Engineering Body of Knowledge (SWEBOK), un proyecto auspiciado por la IEEE para lograr un consenso mundial de lo que es esta disciplina y el lugar que tiene junto a otras ingenierías. El SWEBOK define diez Knowledge Areas (KAs) o áreas de conocimiento."
Sofware Gururu
